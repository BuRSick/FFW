# Google Sheets Setup

Ниже рабочий вариант именно для вашей таблицы:

- [Google Sheet](https://docs.google.com/spreadsheets/d/1Qcg1lUG1Sh8MQgsXM80bXgqZduAbpn97DcK-u6aEjv4/edit?gid=1102006311#gid=1102006311)

## Что нужно сделать сейчас

1. Откройте ваш проект `Apps Script`.
2. Полностью удалите текущий код из `Code.gs`.
3. Вставьте новый код из файла:
   - [google-apps-script.gs](C:/Users/RPUser/Desktop/wedding-race-3d/FFW/docs/google-apps-script.gs)
4. Сохраните файл.

## Почему это важно

В новом коде таблица открывается не через `getActiveSpreadsheet()`, а напрямую по вашему `SPREADSHEET_ID`.

Это надежнее, потому что Web App может выполняться не в том “активном” документе, который вы ожидаете.

## Как заново опубликовать

1. Нажмите `Deploy`
2. Выберите `Manage deployments`
3. Откройте текущий `Web app`
4. Нажмите `Edit`
5. Подтвердите новый deploy

Проверьте, чтобы настройки были такими:

- `Execute as`: `Me`
- `Who has access`: `Anyone`

## Что должно появиться в таблице

Колонки:

1. `Имя`
2. `Придет/Не придет`
3. `Предпочтения по напиткам`
4. `Предпочтение по еде`
5. `Дата ответа`

## Как быстро проверить

1. Откройте приглашение
2. Выберите напитки
3. Выберите блюдо
4. Нажмите `Да, буду` или `Не смогу`
5. Подождите 3-5 секунд
6. Обновите Google Sheet

Если все настроено правильно, появится новая строка.

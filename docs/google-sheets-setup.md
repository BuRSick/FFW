# Google Sheets Setup

Ниже самый простой вариант, чтобы ответы гостей всегда были под рукой.

## Что получится

Таблица Google Sheets с колонками:

1. `Имя`
2. `Придет/Не придет`
3. `Предпочтения по напиткам`
4. `Предпочтение по еде`
5. `Дата ответа`

## Как настроить

1. Создайте новую таблицу в Google Sheets.
2. Назовите ее, например: `Wedding RSVP`.
3. В таблице откройте:
   - `Расширения`
   - `Apps Script`
4. Удалите содержимое файла `Code.gs`.
5. Скопируйте код из файла:
   - [google-apps-script.gs](C:/Users/RPUser/Desktop/wedding-race-3d/FFW/docs/google-apps-script.gs)
6. Вставьте его в `Code.gs`.
7. Нажмите `Deploy -> New deployment`.
8. Выберите тип:
   - `Web app`
9. В параметрах укажите:
   - `Execute as`: `Me`
   - `Who has access`: `Anyone`
10. Нажмите `Deploy`.
11. Скопируйте `Web app URL`.

## Что подставить в проект

Откройте файл:

- [api.js](C:/Users/RPUser/Desktop/wedding-race-3d/FFW/src/services/api.js)

И замените:

```js
const endpoint = 'YOUR_GOOGLE_SCRIPT_URL';
```

на ваш URL из деплоя Apps Script.

## Как проверить

1. Откройте приглашение.
2. Выберите:
   - придет / не придет
   - напитки
   - еду
3. Нажмите кнопку ответа.
4. Откройте таблицу `RSVP` в Google Sheets.
5. Убедитесь, что появилась новая строка.

## Что важно

- если вы публикуете новый deploy URL, его нужно снова вставить в `api.js`
- таблицу лучше хранить в основном Google-аккаунте, к которому у вас всегда есть доступ
- это лучший быстрый вариант для свадьбы, потому что таблицу удобно смотреть и с телефона, и с ноутбука
